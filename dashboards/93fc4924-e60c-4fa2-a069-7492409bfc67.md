## Sections

### Inbox (count)
```exoql
SELECT (COUNT(?x) AS ?count) WHERE {
  ?x a gtd:InboxItem .
  ?x ems:Effort_status ?s . ?s a ems:EffortStatusBacklog .
}
```

### Next Actions
```exoql
SELECT ?x ?label WHERE {
  ?x a gtd:NextAction .
  ?x exo:Asset_label ?label .
  ?x ems:Effort_status ?s . ?s a ems:EffortStatusDoing .
} ORDER BY ?label
```

### Waiting For
```exoql
SELECT ?x ?label ?delegatee WHERE {
  ?x a gtd:WaitingFor .
  ?x exo:Asset_label ?label .
  ?x gtd:Effort_delegatee ?delegatee .
} ORDER BY ?delegatee
```

### Jedi Ratio (green vs red this week)
```exoql
SELECT
  (COUNT(?green) AS ?proactive)
  (COUNT(?red) AS ?reactive)
WHERE {
  { ?green a gtd:Proactive . ?green ems:Effort_endTimestamp ?t . FILTER(?t > $thisWeekStart) }
  UNION
  { ?red a gtd:Reactive . ?red ems:Effort_endTimestamp ?t . FILTER(?t > $thisWeekStart) }
}
```
