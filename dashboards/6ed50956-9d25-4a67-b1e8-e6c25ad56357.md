## Sections

### Projects without Next Action
```exoql
SELECT ?project ?label WHERE {
  ?project a ems:Project .
  ?project exo:Asset_label ?label .
  ?project ems:Effort_status ?s . ?s a ems:EffortStatusDoing .
  FILTER NOT EXISTS {
    ?task ems:Effort_parent ?project .
    ?task a gtd:NextAction .
    ?task ems:Effort_status ?ts . ?ts a ems:EffortStatusDoing .
  }
}
```

### Stale Waiting For (> 7 days)
```exoql
SELECT ?x ?label ?delegatee WHERE {
  ?x a gtd:WaitingFor .
  ?x exo:Asset_label ?label .
  ?x gtd:Effort_delegatee ?delegatee .
  ?x exo:Asset_createdAt ?created .
  FILTER(AGE_DAYS(?created) > 7)
}
```

### Someday/Maybe to Review
```exoql
SELECT ?x ?label WHERE {
  ?x a gtd:SomedayMaybe .
  ?x exo:Asset_label ?label .
} ORDER BY ?label
```

### This Week's Completed
```exoql
SELECT ?x ?label WHERE {
  ?x ems:Effort_endTimestamp ?end .
  ?x exo:Asset_label ?label .
  FILTER(?end > $thisWeekStart)
} ORDER BY DESC(?end)
```
