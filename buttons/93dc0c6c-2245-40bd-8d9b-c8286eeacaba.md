# Someday/Maybe

Not now, but maybe later. Park a task in the Someday/Maybe list. Transitions from `gtd__InboxItem` or `gtd__NextAction` to `gtd__SomedayMaybe` and sets status to Backlog.

- **Precondition**: Visible when asset has class `gtd__InboxItem` or `gtd__NextAction`
- **Grounding**: Remove InboxItem/NextAction class, add SomedayMaybe class, set status Backlog
- **Icon**: clock (Lucide)
- **Category**: gtd
