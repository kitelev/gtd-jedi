# Delegate

Delegate a task to someone. Transitions from `gtd__InboxItem` or `gtd__NextAction` to `gtd__WaitingFor` and sets status to Doing.

- **Precondition**: Visible when asset has class `gtd__InboxItem` or `gtd__NextAction`
- **Grounding**: Remove InboxItem/NextAction class, add WaitingFor class, set status Doing
- **Icon**: user (Lucide)
- **Category**: gtd
