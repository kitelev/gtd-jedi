# Delegate grounding

Composite grounding for the "Delegate" GTD command:

1. Remove existing `exo__Instance_class` (clears InboxItem/NextAction)
2. Set `exo__Instance_class` to `ems__Task` + `gtd__WaitingFor`
3. Set `ems__Effort_status` to Doing

Note: Setting the delegatee (`gtd__Effort_delegatee`) from user input is not yet supported by the grounding system and requires a future `service_call` implementation.
